## Party Member Learning Exchange Platform

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/2a49a37d2eed4c0689c84a6c089c1aec/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Teacher, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Exam Paper Management
5. Party Member Forum
6. Party Member Announcements
7. Course Information
8. Material Information
9. Party Member Assignments
10. Personal Center
11. Check-in
12. Party Member Assignment Management
13. Exam Records
14. Error Question List


**Teacher Functions:**
1. Login
2. Register
3. Change Password
4. Personal Information
5. User Management
6. Exam Paper Management
7. Question Management
8. Exam Records
9. Error Question Book
10. Assignment Management
11. Assignment Submission Management
12. Course Management
13. Check-in Management
14. Material Information Management
15. Carousel Management



**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Administrator Management
4. Teacher Management
5. User Management
6. Exam Paper Management
7. Question Management
8. Exam Records
9. Error Question Book
10. Assignment Management
11. Assignment Submission Management
12. Course Management
13. Check-in Management
14. Material Information Management
15. Forum Management
16. Announcement Management
17. Basic Data Management (Course Type Management, Subject Management, Material Information Type Management, Paper Assembly Method Management, Assignment Type Management, Announcement Type Management)
18. Carousel Management
